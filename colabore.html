Colabore

Caso você tenha encontrado um erro ou uma oportunidade de melhoria em algum post, como dados, screenshots, formatação, etc, ou queira contribuir com um novo post. Existem duas maneiras principais para colaborar com o jogosdaqui:
Se você é um desenvolvedor
Acesse o GitHub do jogosdaqui e veja como colaborar via pull request.
Se você não é um desenvolvedor
Se você não tem conhecimentos de programação ou HTML, você ainda pode colaborar abrindo um issue no GitHub do jogosdaqui, assim algum desenvolvedor que colabora com o projeto poderá atender o issue que você abriu.
